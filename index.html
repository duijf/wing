Wing documentation

Wing is the CSS toolkit for our branding. It features a mostly black and white and no-bullshit look.

Typography

Wing comes with some sane typographic defaults. Here's what wing does for you:

  • Wing comes with a baseline-grid. Each element aligns on a multiple of 27 pixels.
  • The body font is Fira Sans, from Mozilla.
  • Monospace is handled by Source Code Pro.
  • HTML tags behave as you'd expect.

Headers

The default headers look like this:

h1. 111 / 135px

h2. 62 / 81px

h3. 34 / 54px

h4. 19 / 27px bold

h5. 19 / 27px uppercase
h6. 19 / 27px italic

We still have to make these be a bit more responsive on smaller displays.

Body copy

Paragraph text, lists and similar elements have a maximum length of 760px. You are free to override this, although it won't look as good. Enjoy the whitespace you gain.

The body copy is set at 19px with a line height of 27px. These are calculated through rems. If a user is trying to change the font-size, they can.

Inline elements

  • Of course, we can emphasize with the strong and em tags.
  • <code> tags are rendered in monospace with a dark background.
  • Deleted text is supported through the <del> tag.
  • You can highlight text using the <mark> tag.

Buttons

To make a button, add the .w-btn class to an anchor element.

Example

Button Primary button Succes button Danger button 

<a class="button" href="#">Button</a>
<a class="button primary" href="#">Primary button</a>
<a class="button succes" href="#">Succes button</a>
<a class="button danger" href="#">Danger button</a>

Grid system

The grid system is quite simple. To begin a grid, use the .grid class on any block element you like. Add a nested element with the classes .unit and one for size. Size dictates the width of the element and is one of the following:

.whole
.half
.half
.one-third
.two-thirds
.one-quarter
.three-quarters
.one-fifth
.four-fifths
.two-fifths
.three-fifths
.golden-large
.golden-small

Code for this example

<div class="grid">
  <div class="unit whole">
    <pre>.whole</pre>
  </div>
</div>
<div class="grid">
  <div class="unit half">
    <pre>.half</pre>
  </div>
  <div class="unit half">
    <pre>.half</pre>
  </div>
</div>
<div class="grid">
  <div class="unit one-third">
    <pre>.one-third</pre>
  </div>
  <div class="unit two-thirds">
    <pre>.two-thirds</pre>
  </div>
</div>

etc..
