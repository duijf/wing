Wing documentation

Wing is the CSS toolkit for our branding. It features a no-frills and a no bullshit look. This is the first attempt at documentation.

Typography

Wing comes with some sane typographic defaults, a nice typeface and a baseline grid. Here's what wing does for you:

  • We set up a baseline grid. Every element on this page has a height that is an exact multiple of 27 pixels.

Buttons

There are two kinds of buttons. Normal and large. To make a button, add the .w-btn class to an anchor element. Add .w-btn-large if you want a larger button.

Button large Button

Code for this example

<a class="w-btn w-btn-large" href="#">Button large</a>
<a class="w-btn" href="#">Button</a>

Grid system

The grid system is quite simple. To begin a grid, use the .grid class on any block element you like. Add a nested element with the classes .unit and one for size. Size dictates the width of the element and is one of the following:

.whole
.half
.half
.one-third
.two-thirds
.one-quarter
.three-quarters
.one-fifth
.four-fifths
.two-fifths
.three-fifths
.golden-large
.golden-small
