Wing documentation

Wing is the CSS toolkit for our branding. It features a no-frills and a no bullshit look. This is the first attempt at documentation.

Typography

Wing comes with some sane typographic defaults, a nice typeface and a baseline grid. Here's what wing does for you:

  • We set up a baseline grid. Every element on this page has a height that is an exact multiple of 27 pixels.
  • The body font is Fira Sans, from Mozilla.
  • The default HTML5 tags behave as you'd expect.

Headers

The default headers are, with a scale factor of 1.8 for every larger heading:

h1. 111 / 135px (page titles)

h2. 62 / 81px (main sections)

h3. 34 / 54px (subsections)

h4. 19 / 27px (subsubsections)

Body copy

The body copy is set at 19px as well, with a line height of 27px. All of these are the defaults with the font-size on the root html tag set at 16px. The units are rems and scale accordingly.

Body text, and lists and similar elements have a maximum length of 760px.

Buttons

To make a button, add the .w-btn class to an anchor element.

Button Primary button Succes button Danger button

Code for this example

<a class="w-btn" href="#">Button</a>
<a class="w-btn w-btn-primary" href="#">Primary button</a>
<a class="w-btn w-btn-succes" href="#">Succes button</a>
<a class="w-btn w-btn-danger" href="#">Danger button</a>

Grid system

The grid system is quite simple. To begin a grid, use the .grid class on any block element you like. Add a nested element with the classes .unit and one for size. Size dictates the width of the element and is one of the following:

.whole
.half
.half
.one-third
.two-thirds
.one-quarter
.three-quarters
.one-fifth
.four-fifths
.two-fifths
.three-fifths
.golden-large
.golden-small

Code for this example

<div class="grid">
<div class="unit whole">
<pre>.whole</pre>
</div>
</div>
<div class="grid">
<div class="unit half">
<pre>.half</pre>
</div>
<div class="unit half">
<pre>.half</pre>
</div>
</div>
<div class="grid">
<div class="unit one-third">
<pre>.one-third</pre>
</div>
<div class="unit two-thirds">
<pre>.two-thirds</pre>
</div>
</div>
